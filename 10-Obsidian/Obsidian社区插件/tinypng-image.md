## 概述

Obsidian TinyPNG 插件是一个集成 TinyPNG 图像压缩服务的 Obsidian 插件，可以帮助用户压缩图片以节省存储空间。用户只需按照 README 中的安装和配置步骤，提供 API 密钥和并发选项，即可使用该插件压缩图片。使用插件时，用户只需在 Obsidian 中打开命令面板，搜索“Compress Images”并选择该选项，插件将使用提供的 API 密钥和并发选项开始压缩图片。压缩完成后，用户将收到包含结果的通知。需要注意的是，在使用插件之前，请确保仔细阅读 TinyPNG 的服务条款和 API 使用限制。
