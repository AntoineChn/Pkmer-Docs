## 概述

该插件提供了文件夹笔记的核心功能和 API，使用户可以轻松创建文件夹笔记、设置偏好和模板支持。文件夹和文件夹笔记同步工作，可实现文件夹名称和位置的同步。适用于需要删除/创建文件夹笔记的用户，以及需要文件资源管理和文件夹概览功能的高级用户。开发者可通过 npm 安装并使用 API，实现插件的扩展功能。兼容 Obsidian v0.13.24+。可通过 Obsidian 插件商店或 GitHub 安装。
