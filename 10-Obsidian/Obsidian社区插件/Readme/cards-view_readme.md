# 黑曜石卡片视图插件

这是一个为 Obsidian.md 设计的插件，可以显示您的笔记的卡片视图。

![截图](https://cdn.pkmer.cn/covers/cards-view_2_0.png!pkmer)

## 特点

- 以卡片视图显示笔记
- 按最后修改或创建日期排序

## 安装

1. 导航到 Obsidian 的设置
2. 点击“第三方插件”
3. 关闭安全模式
4. 点击“浏览”并搜索“Obsidian Cards View Plugin”
5. 点击安装
6. 在“已安装插件”选项卡中切换插件打开状态
