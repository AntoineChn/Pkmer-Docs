# 如何安装

## 来自Obsidian的插件市场【尚未】
💜：直接从Obsidian市场安装。

🚗：将`Quorafind/obsidian-sakana-sakanaWidget`添加到BRAT。

## 手动下载

🚚：下载最新版本。解压并将三个文件（main.js、manifest.json、styles.css）放入文件夹 `{{obsidian_vault}}/.obsidian/plugins/Obsidian-Sakana-Widget`。
