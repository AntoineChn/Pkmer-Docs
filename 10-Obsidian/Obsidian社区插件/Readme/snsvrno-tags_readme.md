## v0.1.5 特性

- 隐藏标签上的“＃”。
- 隐藏所有嵌套标签的父级。
- 隐藏所有父级标签。
- 根据模式隐藏标签的部分内容。
- 为每个缩短的标签（根据定义）添加CSS类。
- 根据父级为每个标签添加CSS类。
- 为所有标签添加CSS类。
