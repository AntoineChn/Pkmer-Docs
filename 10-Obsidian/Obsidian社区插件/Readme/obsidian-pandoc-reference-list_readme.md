## Obsidian Pandoc参考文献列表

在当前文档中的每个pandoc引用键处，侧边栏显示一个格式化的参考文献。

设置说明：
- 确保已安装[Pandoc](https://pandoc.org/)。**此插件要求至少2.11版本**。
- 提供一个兼容的参考文献文件路径
- （可选）提供一个兼容的[CSL样式](https://citationstyles.org/)的路径或URL
