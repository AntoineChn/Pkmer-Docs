# 用于 Obsidian.md 的餐饮规划和食谱管理插件

该插件帮助您管理食谱清单，并为一周的餐饮计划。

## 特点

- 根据您拥有的食材搜索和过滤食谱
- 为您需要计划的每周创建/更新或生成“膳食计划”备忘录
- 为您生成本周的购物清单

## 用法

要能够搜索食谱，食谱必须位于相同的文件夹中（默认为 Meals），并且采用以下格式

```md
# 食材
- 1 茶匙 盐
- ...
