## 狡猾

这个黑曜石插件使用户能够将工具提示附加到画布上的每个节点。此外，它还提供了节点之间无缝的导航。

#### 特点

- ###### 节点提示

    ![](https://cdn.pkmer.cn/covers/crafty_2_0.gif!pkmer)

- ###### 导航

    确保设置热键 `设置 > 快捷键` 以便能够导航节点

    ![](https://cdn.pkmer.cn/covers/crafty_2_1.gif!pkmer)

| 热键             | 动作          |
| ---------------- | ------------- |
| 默认为空        | 上一个节点    |
| 默认为空        | 下一个节点    |
