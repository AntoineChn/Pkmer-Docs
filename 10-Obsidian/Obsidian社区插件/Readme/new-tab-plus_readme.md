# 新标签页 +

## 主要特点

允许将 markdown 文件、图形和画布以默认行为在新标签页中打开。

## 可选功能

打开文件或画布时，如果文件的版本已经在您的标签页中打开，请将焦点放在文件的版本上。 （可切换）
