# ManicTime

ManicTime 是一个开源的 Obsidian 插件，它将活动文件的路径发送到本地安装的 ManicTime 客户端。

## 安装

1. 在 Obsidian 中，点击 `设置` → `社区插件` → `浏览`。
2. 搜索 `manictime`，点击 ManicTime 插件。
3. 点击 `安装` 按钮。
4. 点击 `启用` 按钮。
5. 使用 Obsidian，你的活动文件将显示在你的 [ManicTime 客户端](https://manictime.com) 上。
