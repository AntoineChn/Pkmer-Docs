# 黑曜石 - 连续模式

将所有打开的笔记（“叶子”）显示在一个标签组中，作为一个连续可滚动的文档（有时被称为“Scrivenings 模式”）。

## 特点

 - 通过上下文菜单切换连续模式。
 - 通过命令面板在活动标签组中切换连续模式。
 - 在启动 Obsidian 时，恢复连续模式在 Obsidian 关闭时处于连续模式的标签组中。
 - 使用箭头键在笔记之间导航。
 - 通过标签拖放重新排序笔记。
 - 通过命令面板在活动标签组中显示/隐藏笔记标题，以创建更“无缝”的体验。
 - 点击标签时，笔记会滚动到视图中。
