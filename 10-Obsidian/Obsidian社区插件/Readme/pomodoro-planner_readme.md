# 黑曜石番茄计划

黑曜石番茄计划是一个简单的插件，根据番茄工作法生成每日计划，与黑曜石日计划插件兼容。

## 如何使用

1. 安装插件
2. 打开命令面板并运行 `Pomodoro Planner: Generate` 命令
3. 插件将根据输入生成每日计划
4. 您可以将生成的计划插入到光标位置，或将其复制到剪贴板
