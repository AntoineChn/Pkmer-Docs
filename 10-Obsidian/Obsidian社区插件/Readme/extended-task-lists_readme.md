# 扩展任务列表

![image](https://cdn.pkmer.cn/covers/extended-task-lists_2_0.png!pkmer)

### 特点

- 使用自己的 UI 呈现进行中和不会执行的任务项目
  - 进行中的语法：`- [.] 进行中`
  - 不会执行的语法：`- [~] 不会执行`

### 使用方法

只需添加一个进行中或不会做的任务项目，并切换阅读模式（默认为 Ctrl/Cmd + E）！

```markdown
- [ ] 待处理
- [.] 进行中
- [~] 不会做
- [x] 已完成
```
