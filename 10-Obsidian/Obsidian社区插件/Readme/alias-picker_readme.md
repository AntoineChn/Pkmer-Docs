# 别名选择器

Obsidian 的别名选择器插件引入了两个命令：' 选择别名 ' 和 ' 选择块 '。

这些命令简化了在笔记中修改 wiki 或 Markdown 链接的过程。

## 特点

选择别名：轻松从文件的可用别名中为当前链接选择一个替代别名。

选择块：使用“#^BlockIdentifier”语法修改文件中引用的块。

## 如何使用

将光标放在链接上，从插件菜单或通过键盘快捷键执行“选择别名”或“选择块”命令。

从可用的块或别名中选择以进行所需的更改。
