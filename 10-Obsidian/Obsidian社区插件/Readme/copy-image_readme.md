# 黑曜石复制图片插件

通过右键单击图像，轻松将图像从黑曜石复制到剪贴板。

## 特点

- **右键复制**：只需右键单击任何图像即可“复制图像”。图像将被复制到剪贴板中，随时可以粘贴到任何地方。

## 手动安装

1. 在 Obsidian 中，打开 设置 > 第三方插件。
2. 关闭安全模式。
3. 点击 浏览社区插件。
4. 搜索 "Copy Image"。
5. 点击安装。
6. 安装完成后，在已安装插件列表中切换插件开启状态。

## 用法

1. 右键单击图像以将其复制到剪贴板。
