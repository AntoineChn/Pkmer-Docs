# 黑曜石文本焦点插件

在黑曜石中打开新笔记时自动将焦点放在文本区域。

## 用法

安装并启用插件后，每当您在 Obsidian 中打开一个新的笔记时，文本区域将自动聚焦，使您可以立即开始输入而无需点击文本区域。
