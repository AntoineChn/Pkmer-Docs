## 如何使用

1. 在 Obsidian.md 应用程序中打开一个笔记
2. 单击您想要插入经文的行
3. 使用 `cmnd + p` 或 `ctrl + p` 打开命令面板
4. 搜索 "insert verse of the day"
5. 选择 "Verse of the Day: Insert daily verse"
