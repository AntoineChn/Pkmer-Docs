# 圣经链接器专业版

这是一个用于将圣经文本转换为 markdown 链接的 Obsidian 插件。

## 特点

将圣经文本转换为 JW Library 链接。

例如：

ge 1:1

=>

`[Genesis 1:1](jwlibrary:///finder?bible=01001001)`
