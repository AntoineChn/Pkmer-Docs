## 概述

Markdown Tree 插件可使用 Markdown 格式的代码样式（制表符、空格和换行符）创建美观直观的目录树。通过缩进表示嵌套目录，并将文件直接放置在相应的目录下。用户只需按照 README 中的步骤，即可轻松创建更美观的目录树。这个插件适用于需要在 Markdown 文档中展示目录结构的场景，解决了传统目录树不够美观的问题。
